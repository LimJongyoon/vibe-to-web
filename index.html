실습

3-0. Node.js 확인

node --version

v20.18.0 같은 버전이 나오면 → 3-2로 건너뛰기

nodejs.org → LTS 큰 버튼 클릭

  1. .pkg 다운로드 → 더블클릭

    "계속 → 계속 → 동의 → 설치" → Mac 비밀번호 입력

    버전이 나오면 성공 → 3-2로

  1. .msi 다운로드 → 더블클릭 → Next 계속

    "Automatically install the necessary tools"는 체크하지 마세요.
  2. PowerShell 닫고 다시 열기
    node --version

    버전이 나오면 성공 → 3-2로

첫 프롬프트로 웹페이지 만들기

프롬프트 = AI한테 시키는 말

이론
❌ 나쁜 프롬프트
"웹페이지 만들어"
→ 무슨 페이지? 어떤 디자인?
✅ 좋은 프롬프트
무엇을 / 어떻게 / 어떤 형태로
세 가지를 챙긴다
실습 
내 자기소개 웹페이지를 만들어줘.

요구사항:
- 파일은 index.html 하나만
- 제목은 "안녕, 나는 [본인 이름]"
- 짧은 자기소개 한 문단
- 좋아하는 것 3가지 리스트
- 다크모드 디자인 (검은 배경, 밝은 글자)
- 가운데 정렬

CSS는 같은 HTML 파일 안에 <style> 태그로 넣어줘.
💡
권한 창이 뜨면 "Yes, and don't ask again for this session" 선택.
확인
✨
VS Code Explorer에 📄 index.html이 생기면 성공. 내가 안 쓴 코드가 폴더에 진짜로 생겼습니다 — 이게 바이브 코딩의 시작입니다.

보너스 — 스크린샷으로 디자인 보여주기

말로 설명하기 어려운 디자인은 스크린샷을 Claude Code 터미널에 드래그하면 됩니다.

⌘ + Shift + 4 → 드래그 → 바탕화면 PNG 저장 → 터미널에 드래그

Win + Shift + S → 영역 선택 → 파일로 저장 후 터미널에 드래그

💡
이미지만 던지지 말고 "이런 스타일로 내 자기소개 페이지를 만들어줘. 색감과 레이아웃을 비슷하게."처럼 설명을 같이 적으세요.

로컬 서버로 결과 보기

개발 중에는 로컬 서버 — 배포는 Day 3에

이론
로컬 서버 vs 파일 더블클릭
항목파일 더블클릭로컬 서버
주소file:///Users/…http://localhost:5500
자동 새로고침❌ 수동 F5✅ (Live Server)
실제 환경과 유사❌✅

localhost = 내 컴퓨터를 가리키는 별명. 인터넷 없어도 작동.

실습

방법 1. Live Server ⭐ 추천

  1. VS Code 왼쪽 🧩 Extensions 클릭
  2. Live Server 검색

    만든 사람 Ritwick Dey → Install

  1. VS Code에서 index.html 파일 열기
  2. 오른쪽 아래 [ Go Live ] 클릭

    브라우저에서 http://127.0.0.1:5500/index.html 자동 열림

  3. 파일 저장 시 자동 새로고침 ✨
    💡
    종료: 오른쪽 아래 "Port: 5500" 클릭

방법 2. npx serve (터미널)

VS Code 터미널: 메뉴 터미널 → 새 터미널 (자동으로 현재 폴더에서 시작)

npx serve

처음 실행 시 Ok to proceed? (y) → y. http://localhost:3000에서 접속. 종료: Ctrl + C

⚠️
방법 2는 자동 새로고침 없음. F5 (또는 ⌘+R)로 수동 새로고침 필요.
확인
  • 브라우저에서 자기소개 페이지를 봤다
  • Live Server 자동 새로고침이 작동한다
  • 새로고침 단축키를 안다 (F5 / ⌘+R)
🆘
"Go Live" 안 보임 → HTML 파일이 열려 있어야 합니다  ·  디렉토리 목록만 뜸 → 파일명이 index.html인지 확인

터미널 기본 명령어

단축키 요약

Shift + Tab    Plan ↔ Accept 전환
Esc            작업 중단
Esc Esc        체크포인트 복원
↑              이전 명령어
Ctrl+C (×2)    종료
이미지 드래그   스크린샷 인식

슬래시 명령어

/init      CLAUDE.md 생성
/clear     대화 초기화 (새 작업 시작 시)
/compact   대화 압축 (맥락 유지)
/context   토큰 사용량 확인
/resume    이전 세션 복구
/help      도움말
/status    계정 확인
/exit      종료
